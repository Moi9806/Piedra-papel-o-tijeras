# Piedra-papel-o-tijeras
# Descripción
Este proyecto consiste en el desarrollo del juego clásico "Piedra, Papel o Tijera", donde el usuario compite contra la computadora.

#Funcionamiento
1. El usuario elige una opción: piedra, papel o tijera.
2. La computadora genera una opción aleatoria.
3. Se comparan ambas elecciones.
4. Se muestra el resultado: ganar, perder o empate.
